
Proyecto de Currículums Personales
¡Bienvenido al repositorio del proyecto de Currículums Personales de Kevin Araujo y Victoria Quintero! Este proyecto consiste en la creación de dos páginas web en HTML y CSS que muestran los currículums individuales de Kevin Araujo y Victoria Quintero, respectivamente.

Objetivo
El objetivo principal de este proyecto es proporcionar una presentación profesional y atractiva de las habilidades, experiencia laboral, educación y proyectos relevantes de Kevin y Victoria. El diseño cuidadoso y el uso de estilos en CSS aseguran que los currículums sean visualmente agradables y fáciles de leer para los posibles empleadores o cualquier persona interesada en conocer más acerca de ellos.

Contenido del Proyecto
En este repositorio encontrarás los siguientes archivos:

cv-kevin.html: Una página HTML dedicada al currículum de Kevin Araujo.
cv-victoria.html: Una página HTML dedicada al currículum de Victoria Quintero.
Carpeta css: Contiene los archivos de hojas de estilo en cascada (CSS) utilizados para dar formato y diseño a las páginas web.
Cómo visualizar los Currículums
Para acceder a los currículums y visualizarlos, simplemente abre los archivos cv-kevin.html y cv-victoria.html en tu navegador web.
